# Волжские просторы - Турагентство

Веб-сайт турагентства «Волжские просторы» для бронирования экскурсий по Нижнему Новгороду.

## Структура проекта

- `index.html` - Главная страница (темная тема)
- `index-light.html` - Главная страница (светлая тема)
- `registration.html` - Страница регистрации на экскурсию
- `schedule.html` - Расписание экскурсий
- `account.html` - Личный кабинет пользователя
- `tour.html` - Страница описания экскурсии
- `404.html` - Страница "Не найдено"
- `css/` - Стили
- `js/` - JavaScript файлы
- `media/` - Медиа файлы

## Адаптивность

Сайт корректно отображается на всех устройствах:
- Десктоп (1920px+)
- Планшеты (768px - 1199px)
- Мобильные устройства (до 767px)
